## 🔄 相比原项目的改进

### 🎨 UI/UX 重新设计

- **🌟 简约化界面设计**
  - 移除所有工具和分类的 emoji 图标，采用纯文字设计
  - 工具类型标签移至卡片右上角，布局更清晰
  - 卡片尺寸优化，信息密度更合理

- **⚡ 交互体验优化**
  - 移除"全部工具"分类，启动时自动选择第一个分类
  - 新增工具描述 tooltip，鼠标悬停显示详细信息
  - 侧边栏和工具卡片尺寸调整，视觉更舒适

- **🗂️ 拖拽排序功能**
  - 支持分类的拖拽重新排序
  - 支持工具的拖拽重新排序
  - 添加拖拽手柄和视觉反馈
